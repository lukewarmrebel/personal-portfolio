## Critical Anti-Patterns (do NOT repeat these)

- **STALE .next CACHE**: After any manual file changes or `npm run build`, delete `.next/` before `npm run dev` or you'll get 404s on the dev server. ‚Üí Always `Remove-Item -Recurse -Force .next` before starting dev server after non-code changes.
- **PowerShell background processes**: Don't use `npm run dev &` ‚Äî PowerShell doesn't support `&` for backgrounding. Use `Start-Process npm -ArgumentList "run","dev"` or just run dev server in a separate terminal.
- **Apostrophe in filenames**: Notion export filenames use Unicode right-single-quote (U+2019). Use `Where-Object { $_.Name -like "*filename*" }` wildcards, not direct path strings.
